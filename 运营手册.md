# 自媒体自动化系统｜运营手册

> 适用人群：日常做选题、追热点、写公众号文章的运营同学。

---

## 一. 工作台怎么用（最快入口）

入口：左侧菜单 `工作台`

你在这里可以直接“选题 + 出初稿”，适合赶热点/没灵感时快速开写。

### 1）灵感搜索：给主题，直接生成一篇

- 在顶部输入框输入主题（例如“AI 教育”“年底理财”“打工人健康”）
- 按回车
- 系统会自动：全网搜索素材 → 构建框架 → 生成初稿
- 生成成功后会自动跳转到文章详情页（可继续编辑、复制 HTML 去公众号）

### 2）今日灵感：用当天热点一键出稿

- 在 `今日灵感` 区域：
  - 点击某条热点的 `⚡ 一键生成`：直接基于该事件生成文章，并自动跳到文章详情页
  - 点击 `查看详情`：先进入热点详情页，挑选要点/引用/事实后再沉淀到素材包

### 3）今日没有热点怎么办

- 若提示“今日暂无热点榜单”，点击 `生成今日榜单`
- 若生成失败或一直为空，优先去：
  - `数据采集` → `数据源管理` 触发抓取
  - 或 `数据采集` → `抓取记录` 用 `快捷抓取` 先抓一批链接

## 二. 你每天最常用的 3 件事

### 1）找选题（从热点开始）

入口：`数据采集` → `热点榜单`

- 先做采集（很重要）：
  - `数据采集` → `数据源管理` → 选数据源 → `立即触发抓取`
  - 或 `数据采集` → `抓取记录` → `快捷抓取`（粘贴 URL 抓一次）
- 再生成榜单：在 `热点榜单` 点击 `生成今日榜单 (Build)`
- 点进事件 `详情`：看要点/引用/事实/来源

### 2）攒素材（把信息收进“素材包”）

你有两种收口方式（二选一即可）：

- 方式 A（更顺手）：先加到素材篮
  - 在热点详情点 `加入素材篮`
  - 去 `内容生产` → `素材包` → 打开 `素材篮` 抽屉
  - 选择：
    - `从素材篮创建素材包`（新建一个包）
    - 或 `写入到已有素材包`（追加到已有包）
- 方式 B（更直接）：在热点详情点 `写入素材包`
  - 选一个素材包，直接写入

### 3）出稿（生成 → 精修 → 发布/导出）

入口：`内容生产` → `生成软文`

- 关键要填：
  - `主题`（这篇文章讲什么）
  - `关键词`（逗号分隔）
  - `语气/风格`（如：专业但不生硬）
  - `字数`（建议 800~1500 起步）
- 强烈建议：选择一个 `素材包`（文章更像“有依据的内容”，也更可复盘）
- 点击 `开始生成`

生成后：

- 去 `内容生产` → `软文管理`
  - `详情`：复制 `Markdown/HTML`
  - `编辑`：进入编辑器精修（支持 AI 重写/续写/翻译）

发布方式：

- 手动发布（当前最常用）：复制 `HTML` → 粘贴到公众号后台
- 半自动：`内容生产` → `发布` → 创建“草稿箱 draft”（见第 4 节）

## 三. 推荐的日常 SOP（照着做就行）

### SOP-A：追热点出稿（最快）

1. `数据源管理` 先触发抓取（当天必须有采集数据）
2. `热点榜单` → `Build` 生成今日榜单
3. 选 1 个事件进 `详情`
4. `写入素材包`（或 `加入素材篮` 后再落到素材包）
5. `生成软文`：选择素材包 → 生成
6. `软文管理` → `编辑`：改标题、结构、结尾 CTA
7. 复制 `HTML` → 公众号后台排版发布（或创建草稿）

### SOP-B：做资料型/深度稿（更稳）

1. `抓取记录` 用 `快捷抓取` 把核心资料 URL 抓进来
2. 在抓取详情：
   - `抽取并写入素材包`（推荐）
   - 或 `抽取素材到素材篮`（后续再收口）
3. `生成软文` 选择素材包 → 生成
4. `编辑`：把“事实/数据/引用”放在关键段落

### SOP-C：混合打法（热点 + 资料补充）

1. 用 `热点榜单` 定方向
2. 用 `抓取记录/快捷抓取` 补背景、补数据
3. 素材都收进同一个 `素材包`
4. 再去生成与精修

## 四. 你需要知道的 5 个名词（够用版）

- **热点榜单**：当天热点事件列表（先采集、再 Build 才有数据）
- **热点详情**：事件的要点/引用/事实/来源，适合快速取材
- **抓取记录**：你抓过的每个网页/接口的“留档”，可复查正文
- **素材篮**：临时收集区（先选再统一落库）
- **素材包**：最终的“资料包”，生成软文时把它喂给模型

## 五. 发布到公众号（草稿箱 draft）

入口：`内容生产` → `发布`

### 使用前准备

- 先有一篇文章（在 `软文管理` 能看到文章 ID）
- 需要公众号开发配置：`AppID` + `AppSecret`

注意：`AppSecret` 是敏感信息，不要公开、不要提交到仓库。

### 怎么创建草稿

1. 新增发布账号
   - 填名称、平台（目前仅 `wechat_official`）、`appid/secret`
2. 创建草稿任务
   - 选发布账号 + 文章 ID
   - **封面图 URL 必填**（可用上传按钮自动回填）
   - 同步/异步：
     - `同步`：立即返回结果（可能等待稍久）
     - `异步队列`：需要启动 celery worker（没启动可能提示投递失败）
3. 去公众号后台看草稿
   - 内容管理 → 草稿箱 → 二次排版/配图/校对 → 再发布

## 六. 常见问题（优先按这个排查）

### 1）热点榜单为空 / Build 失败

- 先确认今天是否有采集数据：
  - 去 `抓取记录` 看今天有没有新的记录
- 没有就先去：
  - `数据源管理` 触发抓取
  - 或 `抓取记录` 用 `快捷抓取`

### 2）生成失败：Key 无效/超时

- 去 `系统配置` → `API Key 池` 检查 Key 是否启用、是否可用
- 尝试：降低字数/减少一次性生成内容

### 3）抓取预览为空

- 尝试填写 CSS Selector（只抓页面某块区域）
- 或换一个更稳定的来源链接再抓

## 八. 最佳实践（运营角度）

- 优先用 `素材包`：文章更像“有依据的内容”，也方便复盘与追溯来源。
- 标题先定 2~3 个备选：生成后在编辑页快速 A/B 选择。
- 生成只负责“出初稿”：发布前务必做三件事：
  - 检查事实与来源是否可信
  - 加“你的观点/结论/建议”（别做纯搬运）
  - 结尾加 CTA（关注/评论/私信/领取资料等）
